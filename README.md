## Beschrijving
De stekjesbieb is een initiatief van Buurtcampus oost waarin buurtbewoners stekjes kunnen ruilen. De website laat zien welke stekjes er in de kast aanwezig zijn en je kan zelf ook een stekje toevoegen.
